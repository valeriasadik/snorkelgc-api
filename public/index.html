🐟 SnorkelGC API

Cloudflare Worker — datos meteorológicos en tiempo real para spots de snorkel en Gran Canaria.

Endpoints

  • GET /api/spots — condiciones de todos los spots
  • GET /api/spot/:id — condiciones de un spot concreto

Pruébalo

  • /api/spots
  • /api/spot/las-canteras
  • /api/spot/el-cabron

Datos: Open-Meteo · Caché: 1h · App: SnorkelGC
